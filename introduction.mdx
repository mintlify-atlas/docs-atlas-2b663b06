# Quality Hub GINEZ

**Quality Hub GINEZ** is a comprehensive quality management and production control system designed for GINEZ manufacturing operations. Built with Next.js 14 and Supabase, it provides real-time quality monitoring, statistical process control, and multi-location production tracking across 40+ branches.

## What is Quality Hub GINEZ?

Quality Hub GINEZ is an enterprise-grade SaaS platform that streamlines quality control processes for manufacturing operations. It combines statistical process control (SPC) with modern web technologies to deliver:

- **Real-time Quality Monitoring** — Track production quality metrics as they happen
- **Statistical Process Control** — Control charts with automatic conformity classification
- **Multi-location Support** — Manage quality across 40+ production locations
- **Role-based Access Control** — 5 distinct roles with granular permissions
- **Interactive Analytics** — Rich dashboards for FTQ analysis and commercial reports
- **Product Catalog Integration** — Google Sheets integration for dynamic product data

## Architecture Highlights

- **Frontend**: Next.js 14 with App Router, React 18, Tailwind CSS, shadcn/ui
- **Backend**: Supabase (PostgreSQL) with Row Level Security (RLS)
- **Authentication**: Supabase Auth with role-based access control
- **Data Integration**: Google Sheets for product catalog, Google Drive for documentation
- **Visualization**: Recharts for control charts and analytics dashboards

## Who Uses Quality Hub GINEZ?

Quality Hub GINEZ serves multiple user roles across manufacturing operations:

- **Administrators** — Full system access, user management, and configuration
- **Supervisors** — Quality oversight, report generation, and data export
- **Operators** — Production tracking and quality measurement recording
- **Preparers** — Batch preparation and formula reference
- **Consultation Users** — Read-only access to reports and catalogs

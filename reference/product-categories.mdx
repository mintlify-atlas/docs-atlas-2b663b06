## Image Gallery

All product category images are hosted on Imgur and linked in the `PRODUCT_CATEGORIES` constant. These images are displayed in:

- Product catalog navigation
- Bitácora product selection
- Analytics and reporting dashboards
- Commercial analysis drill-downs

## Usage in Application

### Product Selection
```typescript
import { PRODUCT_CATEGORIES } from '@/lib/production-constants'

PRODUCT_CATEGORIES.map(category => (
  <Card key={category.id}>
    <img src={category.image} alt={category.name} />
    <h3>{category.name}</h3>
  </Card>
))
```

### Category Filtering
```typescript
import { CATEGORY_PRODUCTS } from '@/lib/production-constants'

const categoryProducts = CATEGORY_PRODUCTS["Limpiador liquido multiusos"]
// Returns: ["LIMALA", "LIMBOS", "LIMBRI", ...]
```

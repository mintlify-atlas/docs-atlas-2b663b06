## Overview

Appearance standards define the expected visual characteristics of finished products. This is a qualitative assessment performed during quality control.

<Warning>
These standards are extracted from `lib/production-constants.ts:244-483` and represent the source of truth for appearance verification.
</Warning>

## Appearance Types

There are three main appearance classifications:

1. **CRISTALINO** - Clear/transparent liquid
2. **OPACO** - Opaque/non-transparent liquid
3. **APERLADO** - Pearlescent/pearly appearance

## Validation Logic

Appearance validation in the bitacora form:

```typescript
const expectedAppearance = APPEARANCE_STANDARDS[productCode]

if (actualAppearance === expectedAppearance) {
  return 'CONFORME'
} else {
  return 'NO CONFORME'
}
```

<Info>
Appearance is a qualitative assessment and depends on the operator's visual inspection.
</Info>

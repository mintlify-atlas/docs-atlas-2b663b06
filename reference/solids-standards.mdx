## Overview

Solids standards define the acceptable range of solids percentage for each product. These values are used for quality control conformity analysis.

<Warning>
These standards are extracted from `lib/production-constants.ts:100-220` and represent the source of truth for quality control.
</Warning>

## Validation Range

All solids measurements must be between **0% and 55%**.

## Conformity Classification

Solids measurements are classified using control chart logic:

### Specification Limits (Red Lines)
```typescript
const specMin = PRODUCT_STANDARDS[product].min
const specMax = PRODUCT_STANDARDS[product].max
```

### Tolerance Limits (Yellow Lines)
```typescript
const warnMin = specMin * 0.95  // -5%
const warnMax = specMax * 1.05  // +5%
```

### Classification Logic
```typescript
if (value >= specMin && value <= specMax) {
  return 'CONFORME'  // Green - Within specification
} else if (value >= warnMin && value <= warnMax) {
  return 'SEMI-CONFORME'  // Yellow - Within tolerance
} else {
  return 'NO CONFORME'  // Red - Out of tolerance
}
```

<Info>
The ±5% tolerance zone provides a buffer for minor variations before a product is classified as non-conforming.
</Info>
